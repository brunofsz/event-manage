# Event Manage

Este é um projeto **full stack** de gerenciamento de eventos desenvolvido com **React, Axios, Node.js, Express e MongoDB**. O objetivo é permitir o registro de festas/eventos, incluindo nome, descrição, orçamento, serviços e imagens via URL.

## Funcionalidades

* Cadastro de festas/eventos
* Listagem de todas as festas cadastradas
* Edição e exclusão de festas em tempo real
* Escolha de serviços adicionais
* Conexão com API backend via Axios
* Banco de dados MongoDB para armazenamento
* Feedback visual para ações (criação, edição, remoção)
* Navegação entre páginas utilizando React Router DOM para a navbar e rotas

## Tecnologias utilizadas

* React.js
* Axios
* CSS3
* JavaScript (Vanilla + React)
* Node.js
* Express.js
* MongoDB (Atlas)

## Como usar

* Abra o link "[https://event-manage-theta.vercel.app/](https://event-manage-theta.vercel.app/)" no seu navegador

  OU

1. Clone o repositório
2. Acesse a pasta do backend
3. Execute `npm install` e `npm start`
4. Acesse a pasta do frontend
5. Execute `npm install` e `npm run dev`
6. Abra o navegador e utilize o sistema normalmente
